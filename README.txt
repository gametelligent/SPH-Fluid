The following solution file is a Microsoft Visual Studio 2013 file and might not open for older versions.
If the device you are running my code on, does not have this version and cannot open the solution, use the following steps:

1. Create a new blank solution in your version of Visual Studio.
2. Right click on the solution in the solution explorer
3. Go to Add->Existing Project
4. Find and open WaterSimFYP.vcxproj
